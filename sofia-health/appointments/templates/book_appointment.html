API Usage

You can also create appointments via API:

curl -X POST http://127.0.0.1:8000/appointments/api/appointments/ \
  -H "Content-Type: application/json" \
  -d '{
    "provider_name": "Dr. Smith",
    "appointment_time": "2024-02-15T10:30:00",
    "client_email": "patient@example.com"
  }'
